Common questions

Does it need internet at all?
No. The AI model and the offline library both run entirely on the device. No data leaves the box. It works in the middle of the desert.
What can it actually answer?
Practical questions: first aid, plant identification, repairs, recipes, navigation basics, weather patterns. It has access to a full offline Wikipedia and WikiHow. It cannot browse the web.
Why is the Pi version so much cheaper?
The Pi 5 costs less but runs the AI model about 3× slower. For most camping questions, slower is fine — you're not in a hurry. Both give the same answers; one just types faster.
How many phones can connect at once?
Multiple people can be connected to the WiFi, but AI responses are queued — one at a time. The offline library (Wikipedia, WikiHow) handles concurrent readers just fine.
Can I use it for medical emergencies?
SurvivorBox is a reference tool, not a medical device. In a real emergency, call 911 first — the device will remind you of this. It's a backup for when you need information and have no signal, not a replacement for professional care.
What about power in a blackout?
It runs on very low power — the Pi version draws under 10W, the N100 around 15W. A small portable battery bank can run it for hours.
Does it update?
When it's connected to internet (at home, a campground with WiFi, etc.) it can download new domain packs via the admin panel. The core software updates are manual for now.
